 Spark — Dating Application

A full-stack dating application built with React and Firebase featuring 
swipe-based matching, OTP authentication, real-time chat, and profile management.

 🚀 Live Demo
[Live Demo](https://sweet-spark-dating-app.netlify.app)
 ✨ Features
- 🔐 User Authentication (Email/Password + OTP Phone Verification)
- 💘 Swipe-based Profile Discovery
- 💚 Like / Match Detection System
- 💬 Real-time Chat between Matched Users
- 👤 Profile Creation with Photo, Bio & Interests
- 📍 Location-based Discovery
- 🔔 Push Notifications (Firebase FCM)

 🛠️ Tech Stack
- **Frontend:** React.js, Vite, JavaScript (ES6+)
- **Backend:** Firebase (Auth, Firestore, Storage)
- **Styling:** Custom CSS Design System
- **Deployment:** Netlify
